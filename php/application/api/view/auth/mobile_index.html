中国超级模特大赛简介

中国超级模特大赛是中国最顶级、最权威、最专业的服装表演大赛，由中国服装设计师协会职业模特委员会主办，东方宾利文化传媒有限公司承办的全国专业大型模特大赛，代表了中国参赛选手的最高水平。本年度前十名优秀选手将代表中国参加每年一届由亚洲十至十五国联合举办的亚洲超级模特大赛总决赛。该赛事每年上半年举行，现已成为中国优秀职业模特进入亚洲和世界超级模特行列的桥梁和纽带。

中国超级模特大赛背景

中国超级模特大赛是唯一一个在有十五年历史的中国国际时装周上举行的顶级模特赛事，更是中国国际时装周（3月秋冬季）的开幕演出。因此本赛事得到了极大的媒体关注度，时尚界关注度和非同一般的社会影响力。
